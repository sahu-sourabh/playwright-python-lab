# Playwright Python Lab 🚀

A professional automation sandbox for mastering Cross-Browser testing and AI-driven quality engineering.

## 🛠️ Setup
1. Clone the repo.
2. Create venv: `python -m venv venv`
3. Activate: `.\venv\Scripts\activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Install browsers: `playwright install`

## 🧪 Running Tests
- Run all tests: `pytest`
- Run in headed mode: `pytest --headed`
- Run across all browsers: `pytest --browser chromium --browser firefox --browser webkit`
